## Development

### Install

Install [nodejs](https://nodejs.org/en/)

```sh 
# install dependencies
npm i
```

### Dev


```sh 
# start dev on https://localhost:8888
npm run dev
```

### Build


```sh 
# Create minified js version in folder GamePush
npm run build
```
